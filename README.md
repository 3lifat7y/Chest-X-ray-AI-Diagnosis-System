## 🧠 Overview

This project is an AI-powered system for detecting chest diseases from X-ray images.  
It includes a deep learning model based on **ResNet50**, a Flask API backend, and a simple frontend to display predictions and recommend nearby doctors.

---

## 🛠️ Tools & Technologies

| Area        | Tech Stack                          |
|-------------|--------------------------------------|
| AI Model    | Python, PyTorch, ResNet50            |
| Backend     | Flask API                            |
| Frontend    | HTML, CSS (or React if applicable)   |
| Data        | Chest X-ray Dataset + Excel Labels   |
| Utilities   | pandas, sklearn, torchvision         |
